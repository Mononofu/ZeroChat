ZeroChat - a web-based sdc chat

Using this website, you can chat on all sdc chat servers that support the InterServer interface. Additionally, more convenient features are available:

  • /join #channel
    lets you join a specific room - no need to wait for invites. Note: This only works with open rooms on this server or rooms on other servers that have the ChanServ bot.

Join a channel to start chatting!
